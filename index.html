• ¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es una red inmensa de computadoras alrededor del mundo conectadas entre sí. La WEB es una enorme colección de páginas que se asientan sobre esa red de computadoras.

• ¿Cuáles son las partes de un URL?

Protocolo HTTPS, subdominio, dominio, TLD, ruta, parámetro y etiqueta

• ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET: El método GET solicita una representación de un recurso específico. Las peticiones que usan el método GET sólo deben recuperar datos.
HEAD: El método HEAD pide una respuesta idéntica a la de una petición GET, pero sin el cuerpo de la respuesta.
POST: El método POST se utiliza para enviar una entidad a un recurso en específico, causando a menudo un cambio en el estado o efectos secundarios en el servidor.
PUT: El modo PUT reemplaza todas las representaciones actuales del recurso de destino con la carga útil de la petición.
PATCH: El método PATCH es utilizado para aplicar modificaciones parciales a un recurso.
DELETE: El método DELETE borra un recurso en específico.

• ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

Los tags y atributos desaprobados son partes del estándar HTML cuyo uso ya no es recomendado. Esto sucede dado que el estándar es actualizado de vez en cuando (por ejemplo, de la versión 3.0 a la 4.0) y alguno tags o características del lenguaje son agregada, removidas o desaprobadas.

• ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

Una de las principales diferencias es que HTML5 define una sintaxis que es compatible HTML4.
El HTML5 permite incluir elementos SVG y MathML, donde se tienen nuevos elementos como section, article, aside, header, footer, etc…
HTML4 suele rmperse mas a menudo en navegadores modernos, a diferencia de HTML5.
HTML5 soporta elementos multimedia.
HTML5 tiene mejor rendimiento y soporte móvil.

• ¿Qué componentes de estructura y estilo tiene una tabla?

De estructura tiene celdas de cabecera, títulos, agrupación de filas, agrupación de columnas.
De estilo tiene summary, width, frame, rules, border, cellspacing, cellpadding y aling
